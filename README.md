# Stock Echoes Scrapper
I turn ranaroussi/yfinance open source library (https://github.com/ranaroussi/yfinance) into an HTTP serverless API.

## Elements
This repository acts as a datascrapper for Stock Echoes - a financial dashboarding application. Stock Echoes is a system, currently composed by:

Frontend - Next.js (Typescript) - https://github.com/DuarteMCSilva/stock-echoes-doggo-ui
Server - Quarkus (Java) - https://github.com/DuarteMCSilva/stock-echoes-server
Data Scrapper - AWS Lambda (Python) that consumes yfinance library - this project

(documentation is in progress)
